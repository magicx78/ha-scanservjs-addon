# Legal and Forum Sharing Notes

Date: 2026-03-16

This project is an unofficial community add-on.

## What you can usually share

- Repository link to this project
- Setup steps and configuration examples
- Your own screenshots and test results
- Sanitized logs (without secrets/private data)

## What you should not upload

- Proprietary Brother driver packages (`.deb`) or mirrored vendor binaries
- Private data (tokens, internal IP plans, serial numbers, email addresses)
- Third-party copyrighted material you do not own

## Required disclaimer for posts

Use this wording (or equivalent):

`Unofficial community add-on. Not affiliated with Home Assistant or Brother.`

## Brand and trademark care

- Use product names factually
- Do not imply official support/endorsement
- If you use logos/images, keep source/rights clear

## Practical publishing checklist

1. Post link to GitHub repo (not binary mirrors).
2. Mention tested version/runtime (for example `v1.0.13`, `r29`).
3. Add the unofficial-project disclaimer.
4. Remove sensitive data from logs before posting.
5. Refer users to official Brother download sources for proprietary packages.

## Important note

This document is practical guidance, not legal advice.
